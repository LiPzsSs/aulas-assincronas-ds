# Sobre o Repositório
Este repositório contém atividades práticas desenvolvidas durantes as aulas remotas para exercitar conceitos fundamentais da linguagem Python.



# Aulas do dia 03/11/2025

## Atividade 1 – Verificador de Maioridade (`atividade1.py`)

### Descrição
O programa pergunta o nome e a idade do usuário, e então informa se ele é maior ou menor de idade.

### Objetivo
Praticar:
- Leitura de dados com input();
- Conversão de strings para inteiros com int();
- Estrutura condicional if/else;
- Uso básico de classes em Python.


## Atividade 2 – A Tabuada (`atividade2.py`)

### Descrição
O programa solicita um número inteiro ao usuário e exibe sua tabuada de 1 a 10.

### Objetivo
Praticar:
- Estruturas de repetição com for;
- A função range();
- Manipulação e formatação de strings;
- Uso de classes e métodos.

## Atividade 2 – Herança de Veículos (`heranca_veiculos.py`)

### Objetivo
Praticar:
- Conceitos de Programação Orientada a Objetos (POO);
- Criação de classes e **atributos;
- Uso de herança entre classes;
- Implementação de métodos personalizados.

### Descrição
O exercício implementa duas classes:
- Veiculo → classe base (ou “pai”), com atributos gerais de um veículo (como marca e modelo).  
- Carro → classe filha que herda de Veiculo e adiciona novos atributos específicos (como cor e número de portas).

Cada classe possui um método exibir_informacoes() que retorna uma string formatada com os dados do objeto.  
No final do programa, são criados dois objetos Carro, e suas informações são exibidas na tela de forma organizada.
